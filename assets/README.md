# Логотип

Положи сюда файл **`logo.ico`** — он автоматически станет иконкой:
- окна приложения и значка в системном трее;
- `.exe` и ярлыков (меню «Пуск» / рабочий стол);
- установщика Velopack (`Setup.exe`).

Ничего больше делать не нужно — сборка (`dotnet build`) и релиз
(`build\release.ps1`) подхватывают `assets\logo.ico`, если он есть.

## Требования к файлу
- Формат: **ICO** (а не png/jpg). Имя ровно `logo.ico`.
- Желательно мультиразмерный: 16, 32, 48, 256 px (квадрат).

## Как сделать .ico из картинки
- Онлайн-конвертер (проще всего): загрузи квадратный PNG → получи ICO с набором размеров.
- Или положи сюда `logo.png` (квадрат, 256×256+) и попроси собрать `logo.ico` из него.
